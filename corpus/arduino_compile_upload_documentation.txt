### Steps to Compile and Upload an Arduino Sketch


1. **Identify the Correct Board**:
List all the connected Arduino boards to identify the correct Fully Qualified Board Name (FQBN).
```shell
arduino-cli board list
```

2. **Compile the Sketch**:
Use the specific FQBN for the Arduino board to compile the sketch.
```shell
arduino-cli compile --fqbn arduino:renesas_uno:unor4wifi Path\\to\\sketch
```

3. **Upload the Sketch**:
Upload the compiled sketch to the Arduino board.
```shell
arduino-cli upload -p COM5 --fqbn arduino:renesas_uno:unor4wifi Path\\to\\sketch
```
